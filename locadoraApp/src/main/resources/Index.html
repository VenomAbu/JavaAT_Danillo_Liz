Bem-vindo as locadoras Rosalina!

Nossa franquia de locadoras existe desde os primórdios das mídias, isso desde a mais antiga galáxia!

Esse projeto foi criado com Spark Java e Maven para criar e implementar uma API, simulando um sistema de gerenciamento de locadoras capaz de criar objetos dos itens que estão no navbar. O projeto foi desenvolvido baseado nas aulas do professor e seguindo suas instruções, usando um MAP para representar um banco de dados e até implementando uma API externa (a viacep). Tudo de bom e se mantenha seguro pelas galáxias!
